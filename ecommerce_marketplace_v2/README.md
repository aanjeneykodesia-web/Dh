# E-Commerce Marketplace V2

Features:
- Buyer, Seller, and Admin roles
- SQLite database (`marketplace.db`)
- Product CRUD
- Product image upload
- Shopping cart (session-based)
- Order placement
- Admin dashboard

## Run
```bash
pip install -r requirements.txt
python app.py
```

Default admin login:
- Email: admin@example.com
- Password: admin123
